1.什么是注解：
        是一种元数据（描述数据的数据） ，应用于类，方法，参数，变量，构造器及包声明的特殊修饰符。
        
2.为什么要引用户注解
   在注解之前,XML被广泛使用描述元数据。
    如果你想为应用设置很多的常量 和参数，这种 情况 下，XML是很好的选择，因为它不会与代码相连，
    如果你想把某个方法声明为服务，那么使用Annotation 会更好一些，这种 情况 需要 注解与方法紧密耦合起来。
 
 目前，许多框架将XML和Annotation两种方式结合使用，平衡两者之间的利弊。
 
3.Annotation是如何工作的？怎么编写自定义的Annotation？
    使用JAVA的反射机制，将类中的注解值取出来。然后实现必要的逻辑
